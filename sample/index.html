Image Jigsaw jquery plugin

Create jigsaw like effects on images with this easy to use jQuery plugin.


how to use?

Wrap up an image inside a div like this: 

<div class="panel" style="display: inline-block">
    <img src="img.jpg">
</div>

Now include jQuery library and this library, and call the jQuery method like: 

<script src="jquery.js"></script>
<script src="jquery.image-jigsaw.min.js"></script>
<script>
$(document).ready(function() {
   $(".panel").jigsaw({freq: 2000, x: 4, y:4, margin: 2});	
});
</script>

Property fields?

Property Description
freq Time in milli seconds when the images should change
x No of columns
y No of rows
margin Margin between each blocks
distinct (new) set to false if you dont require each block to be distinct images
